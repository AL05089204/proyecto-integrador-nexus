# Instalación y configuración

## Prerrequisitos

- macOS con Xcode 15 o superior.
- Cuenta de desarrollador Apple (para ejecutar en dispositivo físico).
- Acceso a la instancia de Payload CMS:
  - URL base (por ejemplo): `https://nexus.midominio.com`
  - Usuario y contraseña de prueba, o token JWT.

## Clonar el repositorio

```bash
git clone https://github.com/AL05089204/proyecto-integrador-nexus.git
cd proyecto-integrador-nexus
open Nexus.xcodeproj   # o .xcworkspace si aplica
