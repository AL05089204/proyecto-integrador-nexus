# Guía de contribución

Gracias por tu interés en contribuir a Nexus App.  
Este proyecto se organiza alrededor de requerimientos funcionales (RF) y tareas en GitHub.

## Flujo de trabajo

1. **Crear un Issue**  
   - Usa la convención: `RF-0X – descripción breve`.
   - Ej: `RF-04 – Modo offline para subir contenido`.

2. **Crear una rama**

```bash
git checkout -b feature/RF-04-modo-offline
